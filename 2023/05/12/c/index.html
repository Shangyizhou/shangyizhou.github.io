BOSS直聘打招呼

您好，我对贵司的岗位很有兴趣。我走的是Linux C++路线， 熟悉 C/C++ 基本语法、内存模型、面向对象编程、熟练掌握 STL 容器和 C++ 11 常用特性。我经常在 Linux 环境下开发程序，能够熟练使用 gcc、gdb、makefile、cmake 等工具开发项目。我有在Linux环境下开发项目的经验，曾使用C++11开发过一个网络库和一个轻量级RPC框架。

我现在离校，可以很快实习到岗，请问可以给您发简历吗？

自我介绍

您好，我叫做尚益州。在河南工业大学就读计算机科学与技术专业，现在是大三。

我走的是 Linux C/C++ 路线，在大一的时候，我主要跟老师学习 C/C++ 基础语法，后来自己对 C++ 方向有一定兴趣，于是在课外就继续看视频、书、博客学习了 STL 容器使用、C++11 新特性。

之后在大一下学期时候购买了阿里云的服务器，于是开始接触了 Linux 的学习，这里主要学习了下 Linux 下的基本命令，Linux 下的环境编程、网络编程还有一部分的多线程编程。在大二时候想为这些知识做一个回顾总结，就做过一个简单的 tinywebserver 项目，过于简单，模块封装不够，耦合性较高。于是后面阅读 muduo 源码以及复习了一些 C11 知识后又在 Linux 环境下使用 C11 实现了一个轻量级网络库，它各个模块耦合性较低且增加了许多功能。那段时间也学习到了 RPC 的基础知识，也在 Linux 环境下使用 muduo、protobuf、zookeeper 封装了一个轻量级的 RPC 框架，用于实现跨进程业务的调用，帮助我更好理解这个 RPC 相关的知识。

我在学校有积极参加一些活动，我大一参加了蓝桥杯算法竞赛，然后获取了省赛二等奖。大二的时候与队友组队参加了互联网+比赛，与团队一起获得了省赛的二等奖。之后参加了字节跳动的青训营活动，与大家一起组队使用 Go 语言开发一个极简抖音的后端，主要负责使用雪花算法生成分布式 ID，使用 jwt 中间件完成认证鉴权模块。我们团队最终取得了字节跳动青训营的万马奔腾奖。

我对技术还是有一定热爱，喜欢写一些技术博客或者笔记，放到 typora 或者语雀里面。方便记录回忆以及发博客。

反问环节

  1. 你们部门在做些什么？主要业务是什么？如果自己很荣幸的能够进入贵部门会负责些什么？因为面试者就是这个部门的，通过他的回答，你也能够了解到这个部门正在做的产品和使用的技术。
  2. 您认为我在哪些方面还存在着不足？这是一个很巧妙的问题，因为它可以从侧面反映出你这次面试的结果。如果面试官带有指导性的回答出了你的不足，你需要补充的知识点，这样就代表你这次面试差不多了，应该是能好好准备二面了；如果面试官直言不讳的说你很差或者基础太弱这样的话，你也就知道凉凉了。
  3. 请问面试官对自己职业规划的建议？面试官大概率是技术大佬或者工作过几年的前辈了，在社会上摸爬滚打了几年，知道的肯定比在校生多。这个问题既表达了对面试官身份的认可，也表现出求职者对当前这份工作得在意程度，并且还能得到技术大佬的分享，怎么看都不是亏本的买卖。
